## 🧭 作ったもの

### 🌤️ お天気＋ToDoアプリ
![お天気＋ToDoアプリ](images/wether-to-do.png)
天気とタスクを一緒に確認できるシンプルなWebアプリです。  
OpenWeather APIとローカルストレージを使って、日々のちょっとした管理を楽にすることを目的に作りました。

- 天気情報の取得・表示（OpenWeather API）
- ToDoの保存（LocalStorage）
- 都市名の保存・再利用

👉 [デモを見る](https://hikaru-taniguchi.github.io/weather-todo-app/)  
👉 [GitHubでコードを見る](https://github.com/hikaru-taniguchi/weather-todo-app)

### 🧺 センタクびより – Weather × Life App
![センタクびより](images/sentakubiyori.png)
「今日は洗たくできるか？」を判断するためのアプリです。  
天気データをもとに、乾きやすさをスコア化しています。
気温・湿度・風速・降水確率から簡単なロジックを組み、時間ごとに比較できるようにしました。

天気によって行動が変わるのに、
タスク管理と分離されている不便さを解消するために開発

- 3時間ごとの天気データ取得
- 独自のスコアロジックの設計・実装
- スマホでも見やすいレイアウト

👉 [デモを見る](https://hikaru-taniguchi.github.io/sentaku-biyori/)  
👉 [GitHubでコードを見る](https://github.com/hikaru-taniguchi/sentaku-biyori)
